## Cloud vs Self-Hosted

| Feature | Cloud Plugin | Self-Hosted GPU |
|---------|-------------|-----------------|
| **Setup Time** | Minutes | Hours |
| **GPU Required** | No | Yes |
| **Data Privacy** | Cloud-processed | On-premises |
| **Scaling** | Automatic | Manual |
| **Cost Model** | Per-use credits | Infrastructure costs |
| **Model Updates** | Automatic | Manual pulls |
