Create lifelike digital avatars that respond to audio in real-time.

## What is BitHuman?

BitHuman SDK lets you build **interactive avatars** that animate realistically from audio input, show dynamic movement with speech, work in real-time at 25 FPS, and integrate into any application.

## Installation

<Steps>
<Step>

### Create environment

```bash
conda create -n bithuman python=3.11
conda activate bithuman
```

</Step>
<Step>

### Install SDK

```bash
pip install bithuman --upgrade
```

</Step>
</Steps>

## API Setup

1. Get your API secret at [imaginex.bithuman.ai](https://imaginex.bithuman.ai/#developer)
2. Download an avatar model from the Community page
3. Set environment variables:

```bash
export BITHUMAN_API_SECRET="sk_bh_1234567890abcdef..."
export BITHUMAN_MODEL_PATH="/path/to/model.imx"
```

## First Avatar

```python
from bithuman import AsyncBithuman

runtime = await AsyncBithuman.create(
    model_path="model.imx",
    api_secret="your_secret"
)

async for frame in runtime.run():
    display_frame(frame)  # Your display logic
```
