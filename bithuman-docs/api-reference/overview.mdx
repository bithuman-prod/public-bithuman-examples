The bitHuman API lets you programmatically create, manage, and interact with avatar agents.

## Base URL

```
https://api.bithuman.ai
```

## Authentication

All requests require the `api-secret` header. Get your API secret from [imaginex.bithuman.ai](https://imaginex.bithuman.ai/#developer).

```http
api-secret: YOUR_API_SECRET
```

## Common Error Format

All errors follow the same structure:

```json
{
  "success": false,
  "error": "ERROR_CODE",
  "message": "Human-readable error description"
}
```

| HTTP Status | Meaning |
|-------------|---------|
| `200` | Success |
| `400` | Invalid request parameters |
| `401` | Invalid or missing `api-secret` |
| `404` | Resource not found |
| `429` | Rate limit exceeded |
| `500` | Internal server error |
